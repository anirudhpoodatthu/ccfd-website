About

This project uses machine learning to detect fraudulent credit card transactions. The model is trained on the Kaggle Credit Card Fraud dataset with SMOTE to handle class imbalance.

Models evaluated: Logistic Regression, Random Forest, SVM, KNN
Best model selected automatically by ROC-AUC score.
